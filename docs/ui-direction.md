## UI Direction

### Visual Concept
紙の設計台と古い観察端末が合体した、アイデアを「？」から「！」へ固定する工房。

### App-specific Expression
方眼紙、トレーシングペーパー、黒い計測パネル、朱色の判定印、鉛筆のラフ線を使う。AIらしい抽象光ではなく、手を動かして企画を整理している感触を前面に出す。

### Avoid
紫青グラデーション、ガラスカード、発光する球、汎用AIチャット画面、過剰なblur、説明LP風ヒーロー。

### Layout
左に開始設定、中央に入力スタジオと3つの質問、右に固定・再生成できる成果物ボードを置く。45秒デモで視線が左から右へ流れる。

### Components
チップ、アイコンボタン、録音・カメラ操作、質問カード、ロック付き結果ブロック、公開準備パネル、エクスポートボタン、保存状態表示。

### Color
生成背景の紙色を基調に、墨色、 muted teal、朱色、真鍮色を使う。単色テーマ化を避ける。

### Typography
日本語は短く自然にする。見出しは太く、補足は小さく、機械翻訳調の説明文を避ける。

### Main Flow
作りたいものを選ぶ → 声・文字・画像・ジェスチャーで材料を足す → 3問に答える → 可視化する → ロック・再生成・Markdown/JSON出力。

# UI Visual Brief

- Visual concept: tactile idea workbench where rough thoughts are pinned into a practical product specification.
- References: grid paper, tracing paper, brass clips, black measurement panels, pencil sketch marks, red-orange approval stamps.
- Color roles: base off-white paper, text charcoal ink, accent muted teal, decision/highlight red-orange, secondary brass.
- Background: wide generated workbench image with central negative space for UI overlays and visible paper texture on mobile crops.
- Control skin: generated textless raster button frame integrated behind the primary action with CSS state overlays.
- Motif: generated raster motif strip integrated into panel headers, plus stamp labels, grid surfaces, history strip, lockable result blocks.
- Text rule: generated image contains no baked-in text; all labels are HTML.
- Crop/contrast: overlay uses warm translucent paper panels and high-contrast ink.
- Avoid: generic AI SaaS gradients, glass cards, glowing particles, robot mascots, decorative blobs.
